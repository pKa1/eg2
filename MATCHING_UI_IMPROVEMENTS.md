## 🎯 Как это работает:

### 1. Преподаватель создает вопрос:
- Вводит 4 пары: левая колонка → правая колонка
- Например: Python → Гвидо ван Россум
- Видит инструкцию и примеры

### 2. Студент проходит тест:
- Видит левую колонку с элементами (A, B, C, D)
- Видит правую колонку с вариантами (1, 2, 3, 4) - **порядок перемешан!**
- Для каждого элемента выбирает соответствующий вариант из dropdown
- Выбор сохраняется автоматически

### 3. Система проверяет:
- Сравнивает выбранные пары с правильными
- Подсчитывает баллы
- Показывает результат
